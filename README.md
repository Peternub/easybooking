# 📅 Система бронирования услуг (Telegram Mini App)

Автоматизированная система онлайн-записи клиентов салона красоты через Telegram.

## 🚀 Возможности

### Для клиентов:
- 📱 Запись на услуги через Telegram Mini App
- 📅 Выбор удобной даты и времени
- 👤 Выбор мастера
- ⏰ Автоматические напоминания за 24 часа и за 1 час
- ❌ Отмена записи в один клик
- ⭐️ Система отзывов (1-5 звезд + комментарий)

### Для администратора:
- 📊 Управление услугами и ценами
- 👥 Управление мастерами
- 📆 Управление графиком работы
- 🔔 Уведомления о новых записях и отменах
- 📝 Просмотр отзывов клиентов

## 🛠 Технологический стек

- **Backend**: Node.js + Grammy (Telegram Bot)
- **Frontend**: React + Vite + TypeScript
- **UI**: @telegram-apps/telegram-ui
- **База данных**: Supabase (PostgreSQL)
- **Календарь**: Google Calendar API
- **Пакетный менеджер**: bun
- **Линтер**: Biome
- **Хостинг**: Vercel

## 📊 База данных

### Основные таблицы:

- **masters** - Мастера салона
- **services** - Каталог услуг
- **master_services** - Связь мастеров и услуг
- **master_schedules** - График работы
- **master_schedule_exceptions** - Отпуска и выходные
- **bookings** - Записи клиентов
- **reviews** - Отзывы
- **admins** - Администраторы

## 🔐 Безопасность

- Row Level Security (RLS) в Supabase
- Авторизация через Telegram WebApp initData
- Проверка подписи данных от Telegram
- Разграничение прав доступа (клиент/администратор)

## 📱 Процесс бронирования

1. Клиент открывает бота и нажимает "Записаться на услугу"
2. Выбирает услугу из каталога
3. Выбирает мастера
4. Выбирает дату и время из доступных слотов
5. Подтверждает запись
6. Получает уведомление об успешной записи
7. Получает напоминания за 24ч и за 1ч
8. Через 2 часа после визита получает запрос на отзыв

## 🔔 Система уведомлений

- **За 24 часа**: Напоминание с деталями записи + кнопка отмены
- **За 1 час**: Короткое напоминание + кнопка отмены
- **Через 2 часа после визита**: Запрос на отзыв с переходом в Mini App

## 🐛 Troubleshooting

### Бот не отвечает
- Проверьте токен бота в переменных окружения
- Убедитесь, что бот запущен
- Проверьте логи в консоли

### Не загружаются данные в Mini App
- Проверьте переменные окружения (VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY)
- Откройте консоль браузера для просмотра ошибок
- Проверьте RLS политики в Supabase

### Ошибки Google Calendar
- Проверьте credentials (Client ID, Client Secret, Refresh Token)
- Убедитесь, что Google Calendar API включен
- Проверьте права доступа к календарям
