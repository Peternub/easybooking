## 🐛 Troubleshooting

Если что-то не работает, проверьте:

1. **Бот не отвечает**:
   - [ ] Правильный токен в `.env`
   - [ ] Бот запущен
   - [ ] Нет ошибок в консоли

2. **Mini App не открывается**:
   - [ ] Правильный URL в кнопке меню
   - [ ] Веб-приложение запущено
   - [ ] Нет ошибок в консоли браузера

3. **Не загружаются данные**:
   - [ ] Правильные ключи Supabase
   - [ ] Применены миграции
   - [ ] Есть тестовые данные
   - [ ] RLS политики настроены

4. **Google Calendar не работает**:
   - [ ] Правильные credentials
   - [ ] Refresh token действителен
   - [ ] API включен в Google Cloud
   - [ ] Календарь существует
