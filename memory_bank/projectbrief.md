# Project Brief

## Название проекта

EasyBooking

## Цель

Собрать MVP для записи клиентов салона через Telegram Mini App с единым backend на Bun и PostgreSQL, а также с админским интерфейсом для операционного управления.

## Рамки проекта

- клиентский сценарий записи на услугу с выбором мастера, даты и времени;
- создание и хранение записей в PostgreSQL;
- Telegram-уведомления о записи, напоминания и запрос отзывов;
- админский интерфейс для управления мастерами, услугами, записями, клиентами и отзывами;
- поддержка ручного добавления записей и промокодов в рамках MVP.
