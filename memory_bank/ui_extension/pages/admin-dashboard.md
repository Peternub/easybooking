# Admin Dashboard

## Route

`/admin-dashboard`

## Назначение

Сводный рабочий экран администратора для календаря, клиентов, отзывов и настроек.

## Основные компоненты

- `CalendarView`
- `ClientsView`
- `ReviewsView`
- `SettingsView`

## Поток данных

- внутри страницы переключаются вкладки с разными операционными представлениями;
- каждая вкладка использует backend API для чтения и обновления данных.
