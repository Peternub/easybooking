# Admin Services

## Route

`/admin-services`

## Назначение

Управление каталогом услуг, категориями, стоимостью и доступностью.

## Основные компоненты

- `ServicesList`
- формы редактирования услуг из `webapp/src/components/admin/`

## Поток данных

- страница читает административный список услуг;
- создание, обновление и переключение активности идут через backend API.
