# Admin Panel

## Route

`/admin`

## Назначение

Точка входа в административный контур Mini App.

## Основные компоненты

- `AdminActionLink`
- `ProtectedAdminRoute`

## Поток данных

- доступ разрешается только после проверки admin access через backend;
- экран служит навигационным шлюзом к остальным административным разделам.
