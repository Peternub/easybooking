# Admin Masters

## Route

`/admin-masters`

## Назначение

Управление справочником мастеров, их профилями, статусами, услугами и графиками.

## Основные компоненты

- `MastersList`
- вложенные формы и секции из `webapp/src/components/admin/`

## Поток данных

- страница загружает список мастеров и связанные административные данные через backend API;
- изменения сохраняются в PostgreSQL через admin endpoints.
