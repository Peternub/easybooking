# Admin Bookings

## Route

`/admin-bookings`

## Назначение

Управление списком записей, просмотр статусов и ручное создание записи администратором.

## Основные компоненты

- `BookingsList`
- формы ручного добавления из административных компонентов

## Поток данных

- страница читает агрегированный список записей;
- ручное создание записи использует административный backend endpoint.
