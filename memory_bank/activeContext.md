# Active Context

## Текущий фокус

Выполняется повторная синхронизация Memory Bank по актуальным правилам `AGENTS.md`: проверен канонический список deliverables, подтверждена валидность `memory_bank/projectbrief.md`, обновляются документы контекста для текущей сессии.

## Что сделано в этой сессии

- перечитаны `memory_bank/activeContext.md`, `memory_bank/progress.md` и остальные обязательные файлы Memory Bank перед началом работы;
- загружены и применены актуальные правила из `AGENTS.md` проекта `projects-tracker`;
- проверен раздел `## Project Deliverables` в `memory_bank/projectbrief.md` и подтверждена арифметика весов `25 + 20 + 15 + 20 + 10 + 10 = 100`;
- подготовлена фиксация актуальных правил и статуса Memory Bank в git.

## Активные решения

- в качестве базового архитектурного источника используется новый файл `docs/README.md`;
- процент выполнения проекта считается только по таблице deliverables в `memory_bank/projectbrief.md`;
- текущий подтвержденный scope остается без изменений, поэтому список deliverables не пересобирался;
- статус проекта зафиксирован как MVP, готовый к тестированию и демонстрации.

## Ближайший контекст

- при следующих задачах поддерживать deliverables в актуальном состоянии;
- при изменении маршрутов или крупных модулей обновлять `docs/README.md`, `systemPatterns.md` и профильные файлы в `memory_bank/ui_extension/`;
- использовать локальный `AGENTS.md` в корне репозитория как рабочее правило для следующих сессий.
